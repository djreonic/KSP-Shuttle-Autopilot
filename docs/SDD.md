# Scope Definition Document (SDD)

## Version
1.2 – DEL Alpha First

## Product Identity
Shuttle-specific deorbit, entry, and landing autopilot.

Not a MechJeb replacement.

## Version 1.0 Scope
- Deorbit planning
- Entry guidance
- TAEM/HAC
- Final approach & autoland
- Scale detection (1×–10×)

## Explicitly Excluded
- Ascent autopilot
- Abort logic
- Rendezvous
- RealFuels
- Failure integrations
- Graphical displays (Beta)

## Governance
No new features during Alpha once freeze declared.
